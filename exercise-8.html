Чередующиеся гласные в корне слова

В корне слова могут чередоваться гласные е//и, а также гласные о//а.

Чередующиеся гласные могут зависеть от разных факторов.

1. От суффикса -а-:

-бер- //-бир- и другие корни с чередованием е//и (всего 9 пар корней);
-кас- // -кос-;
-лаг- // -лож- (но можно также отнести -лаг- // -лож- ко второй группе);
-им- //-а- (-я-);
-ин- // -а- (-я-).

2. От последующей согласной:

-раст-, -ращ- //-рос-;
-скак- // -скоч-.

3. От ударения:

-гар - // -гор-;
-твар-// -твор-;
-клан- //-клон-;
-зар- // -зор-.

4. От значения (смысла) слова:

-мак // -мок-, -моч-;
-равн- // -ровн-.

5. Особое правило действует для корней -плов- // -плыв- // -плав-.

Правописание гласных в корне в случае чередования е//и

По чередованию е//и существует только одно правило:

Есть за корнем суффикс -а- - значит в корне И всегда.

Запомните следующие девять пар корней!

-бер- // -бир-
-пер- // -пир-
-дер- // -дир-
-тер- //-тир-
-мер- // -мир-
-жег- // -жиг-
-блест- // -блист-
-стел- // -стил-
-чет- // -чит-

Примеры:

заберу – забираю

блестеть на солнце – блистать на балу

Исключения: чета (т.е.пара), сочетать, сочетание, словосочетание, бракосочетание. В этих словах гласная а после корня есть, но в корне следует писать букву е.

к оглавлению ▴

Значения корней с чередованием е//и

1) - бер- // -бир-

Значения корней с чередованием: брать, забирать, убирать, выбирать, пробираться и т.п.

Не подходят следующие слова: берег, бережок; беречь, бережно, сбережения, сберегать, сберегательный. В них нет чередования. Гласные проверяемые.

В слове бирюзовый гласная непроверяемая.

2) -пер- // -пир-

Значение корней с чередованием: запирать, выпирать, опираться, препирательства и т.п.

Не подходят слова пир, пировать, перо, перочинный. В них гласные не чередующиеся, а проверяемые.

В словах спираль, перила, эксперимент, темперамент, период, периодический гласные проверить нельзя.

3) - дер- // -дир-

Значение корней с чередованием: выдирать (вырывать), удирать.

Не подходят слова держать, задержать(ся), содержать, содержимое, деревья, деревянный! Гласные не чередующиеся, а проверяемые (держит, содержит, дерево).

В словах директор, директива, гардероб, дирижёр, дирижировать тоже нет чередования. Гласные непроверяемые.

4) -тер- // -тир-

Значение корней с чередованием: вытирать.

Не подходят слова терять, терпение, терпеть, стирать, стирка. Гласные в них не являются чередующимися. Их можно проверить.

В словах тиран, тирания, теребить гласные непроверяемые.

5) - мер- // -мир-

Значения корней с чередованием: замирать, умирать.

Не подходят следующие слова:

- мир, мирный, перемирие, примириться, усмирить, смириться, смирение, непримиримый, умиротворение (в этих словах гласные проверяемые; мир = спокойствие);

- мировой, мировоззрение, мироощущение (в этих словах гласные тоже можно проверить словом мир; мир = Земля);

- мерить (одежду), мерять, примерочная;

- измерять (температуру), измерение, замерять.

Гласные в вышеуказанных словах не чередующиеся, а проверяемые.

Слово меридиан тоже не содержит чередования. Оно словарное (гласная непроверяемая).

6) - жег- // -жиг-

Значения корней с чередованием: жечь, зажигать, выжигать, зажигалка и т.п. (речь идёт об огне, горении).

7) - блест- // -блист-

Значения корней с чередованием: блестеть, блистать.

8) - стел- // -стил-

Значение корней с чередованием: стелить.

В словах стилистический, стилистика, стилизация, стилизовать чередования нет. Гласные проверяются ударением (стиль).

В слове стеллаж гласная непроверяемая.

9) - чет- // -чит-

Значения корней с чередованием: вычитать, почитать (=уважать), почитатель (таланта), сочетать, сочетание и т.п.

Не подходят слова читать, прочитать и т.д. (слова, связанные с чтением)!

Гласные в них не чередующиеся, а проверяемые (прочитывать).

Внимание!

Можно заметить, что самыми простыми для запоминания и узнавания корнями из указанных выше являются - жег- // -жиг-, - блест- // -блист-. Эти корни с чередующимися гласными практически невозможно спутать с проверяемыми и непроверяемыми гласными! Выполняя задание 9 ЕГЭ, обращайте внимание на эти корни. Они могут облегчить для Вас выполнение задания, стать подсказками.

Следует также понимать, что чередование гласных е//и в корнях глаголов обычно соответствует разным видам глаголов – совершенному и несовершенному. Пример: вытирать (несовершенный вид) – вытереть (совершенный вид).

Правописание гласных в корне в случае чередования е//и.

Значения корней с чередованием о//а.

1. Зависят от суффикса –а-.

- кас- //- кос-

Есть за корнем суффикс -а- - значит в корне А всегда.

Примеры: прикоснусь – прикасаюсь, прикосновение, касательная.

Значения корней с чередованием: дотрагиваться (прямой смысл), затрагивать в разговоре (переносный смысл).

Слова косичка, коса, косить (траву) не имеют чередующихся гласных в корне! Гласные проверяются ударением.

А в словах закоснелые (взгляды, общество), касатка, костюм, костюмер, косить (о глазах), косой, косолапый, покоситься, искоса гласные нельзя проверить. Это словарные слова.

- лаг- //- лож-

Есть за корнем суффикс -а- - значит в корне А всегда.

Примеры: приложу – прилагаю, прилагательное, приложение, изложение.

Исключение: пОлог.

Значения корней с чередованием: класть, прилагать, излагать и т.п.

В словах ложь, ложка, логичный, аналогичный чередования нет!

- им- //-а- (-я-)

Есть за корнем суффикс -а- - значит в корне ИМ всегда.

Примеры: принимать – принять, отнимать – отнять, снимать – снять, сжимать - сжать.

Это пары глаголов несовершенного и совершенного вида.

- ин- // -а- (-я-)

Есть за корнем суффикс -а- - значит в корне ИН всегда.

Примеры: начинать – начать, приминать – примять, проклинать – проклясть.

Это пары глаголов несовершенного и совершенного вида.

Обратите внимание на то, что -им- //-а- (-я-), -ин- // -а- (-я-) – это не сами корни, а лишь части корней! Речь идёт о корнях ним–ня, жим–жа, чин–ча, клин–кля.

к оглавлению ▴

2. Зависят от последующей согласной.

- раст-, - ращ- // -рос-

Перед ст, щ пиши А.
Перед с пиши О.

Примеры: вырастать, выращивать, вырос, водоросли.

Перед щ всегда А.

А вот для ст, с есть исключения.

Исключения:

росток, росточек, ростковый и т.п. (т.е. все слова, образованные от слова «росток»)
Ростов, ростовский и т.п. (т.е. все слова, образованные от слова «Ростов»)
ростовщик, ростовщичество и т.п. (т.е. все слова, образованные от слова «ростовщик»)
Ростислав
на вырост
рост
подросток
отрасль, отраслевой, межотраслевой и т.п. (т.е. все слова, образованные от слова «отрасль»)

В слове роскошный чередования нет. Гласную можно проверить (роскошь).

- скак- // -скоч-

Перед к в безударном положении пиши А (СКАК).
Перед ч в безударном положении пиши О (СКОЧ).
Под ударением пиши то, что слышишь
(могут быть корни –скак-, -скок-, -скоч-, -скач-).

Примеры:

скакать (перед к в безударном положении пиши А)
выскочка (перед ч в безударном положении пиши О)
скачки (под ударением пиши то, что слышишь)

Исключения: скачок, скачу, скачи, скачите, скачкообразный.

3. Зависят от ударения.

- гар- // - гор-

Под ударением пиши А (помни слово «загАр»).
Без ударения пиши О.

Примеры: загар, загорелый, загорать, загореть, пригореть.

Исключения: вЫгарки (остатки от сгорания), Изгарь (отходы при плавке руды), прИгарь (привкус гари в подгоревшей пище).

Значения корней с чередованием: гореть, загореть.

Нет чередования в следующих словах:

- алгоритм, горизонт, горизонтальный, горячий (гласные непроверяемые);

- гора, гористый, горный;

- город, городской;

- горе, горевать, огорчение, огорчить, пригорюниться;

- гордость, горделивый, гордиться;

- горький, горечь, горчица, горчичник.

- твар- // - твор-

Под ударением пиши то, что слышишь.
Без ударения пиши О.

Примеры: творить, творец, творение, сотворение, стихотворение, творчество; притворить, притвориться, вытворять, благотворительность, творительный.

Исключение: утварь (без ударения в этом слове буква а).

В словах творог, творожок чередования нет!

- клан- // - клон-

Под ударением пиши то, что слышишь.
Без ударения пиши О.

Примеры: кланяться, поклон (под ударением пишем то, что слышим); склонение, наклонение, преклонение (без ударения пишем О).

В слове клонировать, склон чередования нет!

Таким образом, в корнях -гар- // - гор-, -твар- // - твор-, -клан- // - клон- без ударения пишется гласная О.

- зар- // - зор-

Под ударением пиши то, что слышишь.
Без ударения пиши А.

Примеры: зорька, зарево (под ударением пишем то, что слышим); заря, озарять, зарница, зарянка (без ударения пишем А).

В слове прозорливый чередования нет! Гласную можно проверить (зоркий).

В словах взор, узор, позор, надзор, подзорная (труба) чередования тоже нет.

Также нет чередования в словах озорной, озорничать. Гласные непроверяемые.

4. Зависят от значения (смысла) слова.

- мак // -мок-, -моч-

Слова с корнем – мак - имеют значение «погружать в жидкость». Пример – обмакнуть перо в чернила.

Обратите внимание! Слова промокнуть (что-либо чем-либо), промокашка, промокательная бумага имеют значение «погружать в жидкость», но пишутся через о (-мок-). Это исключения.

Слова с корнями -мок-, -моч- имеют значения «становиться влажным», «пропускать или не пропускать жидкость». Примеры – вымокнуть под дождём, непромокаемая обувь, промочить ноги, мокрица.

В словах макет, макулатура, макароны чередования нет! Гласные в этих словах непроверяемые.

В словах мак, маковый, мочалка, замок, замочный чередования тоже нет.

- равн- // -ровн-

Слова с корнем -равн- имеют значение «равный, одинаковый». Примеры – наравне, уравнение, уравнять в правах, равнозначный, равновесие, сравнить.

Слова с корнем -ровн- имеют значение «ровный, гладкий». Пример – подровнять волосы, выровнять края, разровнять грядку, заровнять яму.

Исключения: равнина, равнение, уровень, поровну, ровесник.

5. Особое правило действует для корней – плов- // -плыв- // -плав-.

Корень -плов- пишется только в словах пловец, пловчиха.

Корень -плыв- - в словах плывуны (слои почвы), плыть, заплыв и т.п..

Корень -плав- - в остальных словах. Примеры – плавать, поплавок, жук-плавунец, плавник, плавучий, сплавляться.

Подсказки для выполнения задания 9 ЕГЭ по русскому языку

Ознакомьтесь с правилами проверки безударных гласных в корне, а затем с перечнем словарных слов, которые часто встречаются в задании 9 ЕГЭ. Чтобы вам было легче запоминать словарные слова с непроверяемыми гласными, мы условно разделили их на группы.

Правила проверки безударных гласных в корне слова

Чтобы проверить безударную гласную в корне, необходимо изменить форму слова либо подобрать однокоренное слово так, чтобы на гласную падало ударение. Это знаменитая орфограмма номер один.

Итак, безударные гласные проверяются ударением: гора – горы, умалять (значение) - мало, умолять (о пощаде) – молит.

1. Нельзя проверять безударную гласную с помощью глагола с суффиксами -ыва-, -ива-. Пример: спросить – спрашивать, опоздать – опаздывать. Видно, что такая проверка сбивает с толку.

2. Однокоренные слова – это слова, которые имеют один и тот же корень с одним и тем же значением. Поэтому нельзя подбирать проверочные слова только по созвучию.

Огромный, громадный, гром – похожие слова. Но они не являются однокоренными! Словом гром нельзя проверить слова огромный, громадный. В них гласные непроверяемые.

Слово стадион нельзя проверить словом стадия. Стадион – словарное слово.

3. Существуют корни, близкие по звучанию, но разные по значению.

Например, отварить (картофель) – отворить (дверь). В первом случае гласную можно проверить словом варка, а во втором – затвор.

Разредить (всходы) – разрядить (обстановку). В первом случае слово редкий будет проверочным. Во втором случае для проверки можно использовать слово разряд.

Слово, стоящее в скобках, помогает понять значение и правильно написать букву. Подобные слова часто встречаются в задании 9 ЕГЭ по русскому языку.

4. Слова акварель, монография, хронология непроверяемые (в русском языке нет слов аква, моно, хроно и т. п.).

5. В слове палисадник нет иноязычного поли, в отличие от слов поликлиника, полифония.

6. Нельзя использовать для проверки искусственно созданные слова. Например, слово экстремальный нельзя проверить словом молодёжного сленга экстрим. Слово компьютер нельзя проверить словом комп.

7. Интеллект интеллигенту необходим, но корни этих слов различаются.

8. В корне иногда происходит чередование согласных, но слова при этом всё равно остаются однокоренными. Примеры: мерцающий – меркнет, просвещение – свет.
